Simple Install
1. Login to WordPress and go to Plugins > Add New > Upload
2. Upload wp-showcase.zip and activate
3. Start adding your galleries in the "Showcase" menu that appears.

Manual Install
1. Upload the wp-showcase folder to your /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Start adding your galleries in the "Showcase" menu that appears.
